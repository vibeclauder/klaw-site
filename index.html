Get Running

Three steps to live trading.

01

Clone & Install

Python 3.11+. Pull the repo, install deps. SQLite works out of the box.

git clone https://github.com/klawtrade/klawtrade && pip install -r requirements.txt
02

Configure Your Broker

Add API credentials to .env. Alpaca, IBKR, Tradier supported. Paper mode available.

cp .env.example .env && nano .env
03

Backtest, then Go Live

Validate with a full backtest. Same code path powers live execution — zero config change.

klaw backtest --strategy momentum && klaw run --live
